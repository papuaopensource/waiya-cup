Panduan Kontribusi

Situs Waiya CUP 2025 ini adalah inisiatif dari penggemar untuk penggemar. Kami sangat menghargai kontribusi dari siapa pun yang ingin membantu menjaga data tetap akurat dan mengembangkan fitur-fitur baru.

1. Kontribusi Data (Admin Panel)

Jika Anda memiliki akses ke panel admin, Anda bisa membantu dengan:

  • Memperbarui Hasil Pertandingan: Setelah pertandingan selesai, pastikan skor dan status pertandingan diperbarui di panel admin.
  • Menambahkan Tim & Pemain Baru: Jika ada tim atau pemain yang belum terdaftar, Anda dapat menambahkannya.
  • Memperbaiki Data Pemain: Memastikan jumlah gol, assist, atau clean sheet pemain akurat.

Untuk mendapatkan akses panel admin, silakan hubungi admin@waiyacup.com (contoh email).

2. Kontribusi Kode (Pengembang)

Kami menyambut baik kontribusi dari pengembang untuk meningkatkan situs ini. Berikut beberapa cara Anda bisa berkontribusi:

  • Laporkan Bug: Jika Anda menemukan masalah atau bug, silakan laporkan melalui repositori GitHub (jika ada).
  • Saran Fitur Baru: Punya ide untuk fitur menarik? Sampaikan saran Anda!
  • Kirim Pull Request: Jika Anda seorang pengembang, Anda bisa mengirimkan Pull Request dengan perbaikan bug atau fitur baru. Pastikan kode Anda bersih, terdokumentasi, dan mengikuti gaya penulisan yang ada.

Repositori GitHub: github.com/your-username/waiyacup-2025 (ganti dengan link repo Anda).

3. Jadi Relawan / Moderator

Jika Anda tertarik untuk membantu dalam kapasitas yang lebih besar, seperti moderasi data, pengelolaan komunitas, atau dukungan, silakan hubungi kami untuk informasi lebih lanjut.

Terima kasih atas dukungan Anda untuk Waiya CUP 2025!
